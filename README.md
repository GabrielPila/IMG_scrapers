# Image Scrapers
      
En este repositorio se guardan los códigos de scrapeo de imágenes desde diferentes fuentes. Las imágenes se pueden utilizar como fuente para proyectos de Computer Vision.

### Dependencias
> Selenium, Pandas
   
| libreria | version |
| --- | --- |
| selenium | latest |
| pandas | latest |
| chromedriver | v90 |
